
# 🔐 Cybersecurity Notes  

## 🧠 XSS (Cross-Site Scripting)  
XSS allows attackers to inject malicious scripts into web applications.

### Example Payload:
<script>alert('XSS')</script>

---

## 💉 SQL Injection  
SQL Injection is used to manipulate database queries.

### Example Payload:
' OR '1'='1

---
